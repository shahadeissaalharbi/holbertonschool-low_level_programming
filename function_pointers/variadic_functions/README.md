# Variadic Functions

## Description
This project explores variadic functions in C — functions that accept
a variable number of arguments using stdarg.h macros.

## Files
| File | Description |
|------|-------------|
| 0-sum_them_all.c | Returns the sum of all parameters passed |
| variadic_functions.h | Header file with all prototypes |

## Compile
gcc -Wall -pedantic -Werror -Wextra -std=gnu89 0-main.c 0-sum_them_all.c -o a

## Example Output
1122
500
